# Draft

Draft is a modern, offline-first productivity web application for writing, organizing, and managing personal information in one place.

It combines notes, to-do lists, journals, projects, and reading lists with support for rich content and file attachments.

---

## ✨ Features

- 📝 Notes with rich text editing
- ✅ To-do list management
- 📁 Projects workspace
- 📖 Daily journal system
- 📚 Reading list tracker
- 📂 Custom folders
- 📎 File attachments (images, videos, PDFs, and more)
- 👤 User profile (name + avatar)
- 🌗 Dark / Light mode toggle
- ⚡ Auto-save functionality
- 💾 Offline-first storage using IndexedDB
- 📱 Fully responsive design

---

## 📎 Attachments

- Supports large file uploads (within browser limits)
- Stores files as Blobs in IndexedDB
- Instant preview for images and videos
- PDF and file download support
- Memory-safe loading with object URL cleanup

---

## 🧠 Key Concept

Draft is designed as a **local-first productivity system**, meaning all data stays on the user’s device without requiring internet or accounts.

---

## 💾 Storage System

- IndexedDB for persistent storage
- Blob-based file storage (no base64 encoding)
- Automatic save and restore on reload
- Memory-optimized attachment handling

---

## 📱 Responsive Design

- Mobile-first layout
- Sidebar transforms into slide-out menu on mobile
- Adaptive editor and note views
- Touch-friendly UI

---

## 🚀 Getting Started

1. Download or clone the project
2. Open `index.html` in any modern browser
3. Start using immediately — no setup required

---

## ⚠️ Limitations

- Storage depends on browser limits
- No cloud sync or multi-device support
- Very large files may still hit device constraints
- Designed for personal/local usage only

---
